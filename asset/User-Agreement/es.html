Acuerdo de Usuario

Bienvenido a nuestra plataforma B2B que conecta distribuidores y minoristas.

Servicios

  • La aplicación de escritorio ofrece gestión de inventario, productos, pedidos, clientes, promociones y reportes, con notificaciones automáticas.
  • La aplicación móvil permite navegar por el catálogo, realizar pedidos, rastrearlos y usar geolocalización.
  • Ambas versiones incluyen chat en tiempo real para soporte y comunicación.

Obligaciones del Usuario

  • Debe proporcionar información precisa al registrarse y usar la plataforma de forma legal.
  • No se permite usar la plataforma para actividades ilegales ni perturbar su funcionamiento.

Descargo de Responsabilidad

La plataforma busca mantener la continuidad del servicio pero no se responsabiliza por interrupciones debidas a fuerza mayor.
